# Unified Page Builder - Quick Start Guide

## What is This?

A **no-code page builder** combining three powerful tools:
1. **Puck** - Visual page builder with 12 block types
2. **Shadon** - Beautiful UI components (Button, Card, Badge, Alert)
3. **AI Assistant** - Smart suggestions to accelerate building

Build professional landing pages, product pages, and marketing sites **without code** and **with AI help**.

## Getting Started

### Step 1: Open the Builder
Navigate to **`/builder/puck-unified`** in your browser.

### Step 2: Choose Your AI Mode

**Top right corner** - Select one of 3 modes:

#### 🎯 **Suggest Mode** (Recommended for beginners)
- Type what you want: "I need a contact form"
- Get 5-7 block suggestions
- Click `+` on any suggestion to add it
- Edit block content in the right panel

#### 💬 **Chat Mode** (Interactive)
- Have a conversation with AI
- You: "Add a navigation bar"
- AI: "✓ Added. What items should it have?"
- Back-and-forth dialogue to build your page

#### 🚀 **Build Mode** (Full page generation)
- Give a description: "Create a landing page for an AI analytics tool"
- AI generates complete page (hero, features, pricing, testimonials, CTA, footer)
- Review and refine

### Step 3: Interact with Suggestions

When suggestions appear on the right:

| Action | Result |
|--------|--------|
| Click `+` button | Adds block to bottom of page |
| Drag block | Drag to sidebar, then drag to canvas at specific position |
| Click block name | More details about the block |

### Step 4: Edit Your Blocks

**Left side (Canvas):** Click any block to select it
**Right panel:** Edit block properties
- Title
- Description
- Button text
- Links
- etc.

### Step 5: Preview Your Page

Your page updates in **real-time** as you build:
- Blocks appear immediately
- Changes are instant
- No save needed

### Step 6: Save Your Work

**Bottom toolbar:**
- **Save Page** - Saves to cloud (share with others)
- **Export** - Downloads JSON file
- **Download** - Downloads as HTML
- **Clear** - Start over (⚠️ This deletes everything)

### How to Remove a Block

1. Click on the block on canvas
2. Press `Delete` key (or look for delete button)
3. Block removed instantly

### How to Reorder Blocks

1. Drag block up/down in canvas
2. Hold and drop at new position
3. Order updates instantly

### How to Edit Block Content

2. Right panel shows editable fields
3. Type or select new values
4. Changes apply in real-time

### How to Get AI Help

4. Pick one or combine multiple blocks

## Tips & Tricks

### Pro Tip 1: Start with Hero + CTA
Every landing page needs a clear headline and call-to-action. Start there!

### Pro Tip 2: Use AI Suggestions for Structure
Let AI suggest the overall page structure (hero, features, pricing, testimonials, cta, footer), then customize each section.

### Pro Tip 3: Edit Before Adding More
Edit one block fully before jumping to add more. This makes better pages.

### Pro Tip 4: Use Keyboard Shortcuts
- `Ctrl+Z` (Cmd+Z on Mac) - Undo last action
- `Ctrl+Shift+Z` (Cmd+Shift+Z) - Redo
- `Delete` - Remove selected block

### Pro Tip 5: Combine Multiple Suggestions
You don't have to pick just one suggestion. Combine 2-3 related suggestions to create rich pages.

## AI Features Explained

### Smart Suggestions
AI understands your request and suggests only relevant blocks:
- "Contact form" → Suggests Contact block (+ others)
- "Pricing page" → Suggests Pricing (+ Navbar, Footer, CTA)
- "Testimonials" → Suggests Testimonial + Carousel

### Deduplication
AI won't suggest blocks you already have. If you've added Hero, it won't suggest Hero again in the same session.

### Context Awareness
AI considers:
- Page type (landing page, product page, blog, etc.)
- Existing blocks
- User intent
- Industry best practices

## Troubleshooting

### "Suggestions not showing"
- Make sure you're in Suggest mode (dropdown at top-right)
- Type a clear prompt and press Enter
- Wait 1-2 seconds for API response

### "Block not appearing"
- Check that you clicked the `+` button (green checkmark)
- Scroll down canvas to see newly added block at bottom
- Try refreshing the page if nothing appears

### "I want to start over"
- Click "Clear" button at bottom
- ⚠️ Warning: This deletes everything! (but you can undo with Ctrl+Z)

### "Where's my saved work?"
- Pages auto-save to browser every 30 seconds
- Check browser localStorage if page doesn't load
- Use "Save Page" button to backup to cloud

### "How do I download a standalone file?"
- Click "Download" button at bottom
- This exports as HTML you can use anywhere
- No internet required to open downloaded file
