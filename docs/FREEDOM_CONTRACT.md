# AI-WONDERLAND – Freedom Contract

AI-WONDERLAND guarantees:

- Structured projects
- Reversible AI actions
- Human-in-the-loop execution
- Safe tool chaining
- Explicit deploy & publish steps

AI-WONDERLAND does NOT guarantee:

- Infinite customization
- Every framework or runtime
- Automatic commits or deploys
- Perfection or completeness

After this boundary, users are free — and responsible.
