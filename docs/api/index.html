    🌌 AI-WONDERLAND

    AI-WONDERLAND is a multi-platform SaaS website & app builder — an advanced web workspace combining a drag-and-drop builder, AI-powered IDE, terminal, and cloud-backed project management.

    It’s designed to feel like a browser-native operating system for building products.

    🚀 Features

    • 🧱 Drag-and-Drop Website & App Builder
    • 🤖 AI-Powered IDE & Assistant
    • 🖥️ Integrated Web Terminal (history + autocomplete)
    • 📦 Project & Workspace Management (shared .data/projects storage)
    • 🤝 Unified AI modules registry with test/apply flows
    • 💾 Snapshots & undo for applied AI changes
    • 👀 Read-only preview from stored project files
    • 🔐 Authentication, Subscriptions & Billing
    • 📊 Analytics & Marketplace Extensions

    🧰 Tech Stack

    • Framework: Next.js (App Router) + TypeScript
    • Styling: Tailwind CSS
    • Auth & Data: Supabase auth; local JSON/file storage under .data/projects
    • State: React Hooks & Context
    • AI: Shared orchestrator + public modules registry (Playground, Wonder Build, WonderSpace)
    • Persistence: File-backed project storage with snapshots/restore and publish artifacts
    • Dev Environment: WonderSpace
